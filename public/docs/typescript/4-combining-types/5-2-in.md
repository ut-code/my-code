### in 演算子

オブジェクトが特定のプロパティを持っているかどうかで型を絞り込みます。

```ts:type-guard-in.ts
type Fish = { swim: () => void };
type Bird = { fly: () => void };

function move(animal: Fish | Bird) {
  if ('swim' in animal) {
    // ここでは Fish 型
    animal.swim();
  } else {
    // ここでは Bird 型
    animal.fly();
  }
}

const fish: Fish = { swim: () => console.log("Swimming...") };
move(fish);
```
