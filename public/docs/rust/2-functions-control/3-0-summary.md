## この章のまとめ

  * **関数:** パラメータの型と戻り値の型を明示する。
  * **文と式:** Rustは式指向であり、セミコロンのない最後の式がブロックの戻り値となる。
  * **if式:** `if` は値を返すことができる。条件は必ず `bool` でなければならない。
  * **ループ:** `loop`（無限・値返し可能）、`while`（条件付き）、`for`（イテレータ・範囲）があり、特に `for` が推奨される。
