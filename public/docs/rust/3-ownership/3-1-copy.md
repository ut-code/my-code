### Copyトレイト：スタックのみのデータ

整数型のような単純な値は、サイズが固定でスタック上にあります。この場合、変数を代入すると**値がコピー**されます。

```rust
let x = 5;
let y = x; // xの値(5)がコピーされてyに入る
// ここでは x も y も両方有効
```
