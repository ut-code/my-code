### 基本的な構文

以下の関数は、引数 `item` が `Summary` トレイトを実装していることを要求します。

```rust
// 糖衣構文（Syntactic Sugar）
fn notify(item: &impl Summary) {
    println!("Breaking news! {}", item.summarize());
}

// 正式なトレイト境界の構文
fn notify_formal<T: Summary>(item: &T) {
    println!("Breaking news! {}", item.summarize());
}
```
