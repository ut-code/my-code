## 5.5 その他のスライス

スライスは文字列だけでなく、配列に対しても使えます。

```rust:array_slice.rs
fn main() {
    let a = [10, 20, 30, 40, 50];

    // 配列の一部を借用する
    let slice = &a[1..3]; // [20, 30]

    // assert_eq! は値が等しいか確認するマクロ
    assert_eq!(slice, &[20, 30]);
    
    println!("Slice elements: {:?}", slice);
}
```

```rust-exec:array_slice.rs
Slice elements: [20, 30]
```
