### 練習問題1: アロー関数への書き換え

以下の関数宣言を、アロー関数 `isEven` に書き換えてください。ただし、省略可能な記号（カッコやreturnなど）は可能な限り省略して最短で記述してください。

```js:practice4_1.js
function isEven(n) {
  return n % 2 === 0;
}
```
