### 基本構文

`function` キーワードを省略し、`=>` (矢印) を使って定義します。

```js:arrow_function.js
// 従来の関数式
const add = function(a, b) {
  return a + b;
};

// アロー関数
const addArrow = (a, b) => {
  return a + b;
};

console.log(addArrow(3, 5));
```

```js-exec:arrow_function.js
8
```
