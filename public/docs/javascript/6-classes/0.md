## クラスの定義とコンストラクタ

JavaScriptのクラスは `class` キーワードを使って定義します。初期化処理は `constructor` という特別なメソッド内で行います。

基本的に、クラス定義の内部は自動的に **Strict Mode (`'use strict'`)** で実行されます。

```js-repl:1
> class User {
...   constructor(name, age) {
...     this.name = name;
...     this.age = age;
...   }
... }
undefined
> const user1 = new User("Alice", 30);
undefined
> user1.name
'Alice'
> typeof User // クラスの実態は関数
'function'
```
