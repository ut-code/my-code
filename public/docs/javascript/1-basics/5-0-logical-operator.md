## 論理演算子と Null 合体演算子

### 論理演算子

JavaScriptには3つの基本的な論理演算子があります。

  * `&&`（論理積 / AND）: 左辺が falsy であれば左辺を返し、そうでなければ右辺を返します。
  * `||`（論理和 / OR）: 左辺が truthy であれば左辺を返し、そうでなければ右辺を返します。
  * `!`（論理否定 / NOT）: 真偽値を反転します。

JavaScriptでは、`false`、`0`、`""`（空文字列）、`null`、`undefined`、`NaN` が **falsy** な値です。それ以外はすべて **truthy** です。

```js-repl
> true && "Hello"
'Hello'
> false && "Hello"
false
> "Alice" || "デフォルト"
'Alice'
> "" || "デフォルト"
'デフォルト'
> !true
false
> !false
true
```

`||` はデフォルト値を設定するときによく使われます。ただし、`0` や `""` のような falsy な値もデフォルト値に置き換わってしまう点に注意が必要です。

### Null 合体演算子 (??)

`??`（Null 合体演算子）は、左辺が `null` または `undefined` の場合にのみ右辺を返します。`0` や `""` は falsy ですが、`??` では右辺に置き換わりません。

```js-repl
> null ?? "デフォルト"
'デフォルト'
> undefined ?? "デフォルト"
'デフォルト'
> 0 ?? "デフォルト"
0
> "" ?? "デフォルト"
''
> 0 || "デフォルト"
'デフォルト'
```

値が `null` または `undefined` のときだけデフォルト値を使いたい場合は `??` を、falsy な値すべてをデフォルト値に置き換えたい場合は `||` を使いましょう。
