### スプレッド構文 (...)

配列を展開する構文です。配列の結合や、\*\*浅いコピー（Shallow Copy）\*\*の作成によく使われます。

```js-repl:3
> const part1 = [1, 2];
undefined
> const part2 = [3, 4];
undefined
> // 配列の結合（新しい配列を作成）
> const combined = [...part1, ...part2];
undefined
> combined
[ 1, 2, 3, 4 ]

> // 配列のコピー（新しい参照を作成）
> const copy = [...part1];
undefined
> copy === part1
false
```
