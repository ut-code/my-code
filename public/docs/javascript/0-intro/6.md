### ブラウザでの実行 (参考)

ブラウザで動かす場合は、HTMLファイルが必要です。
`index.html` を作成し、以下のように記述してブラウザで開いてみてください。

```html
<!DOCTYPE html>
<html>
<body>
  <script>
    console.log("Hello from Browser!");
    alert("Hello from Browser!");
  </script>
</body>
</html>
```

ブラウザの開発者ツール（Consoleタブ）にメッセージが表示され、ポップアップウィンドウが出れば成功です。
