### macOS / Linux

macOSでは、**Homebrew** というパッケージマネージャを使ってインストールするのが簡単です。
`brew install python`

もちろん、Windowsと同様に公式サイトからインストーラをダウンロードすることも可能です。多くのLinuxディストリビューションには初めからPythonがインストールされていますが、最新版を使いたい場合はディストリビューションのパッケージマネージャ（`apt`, `yum`など）を利用するのが一般的です。
