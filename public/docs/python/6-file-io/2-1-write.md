### 書き込み

**`write()`** メソッドは、文字列をファイルに書き込みます。このメソッドは書き込んだ文字数を返します。

```python-repl
>>> f = open('test.txt', 'w', encoding='utf-8')
>>> f.write('こんにちは、世界！\n')
9
>>> f.write('これは2行目です。\n')
9
>>> f.close()
```

`write()` は自動的には改行しないため、必要であれば自分で改行コード `\n` を追加します。

```text-readonly:test.txt
こんにちは、世界！
これは2行目です。
```
