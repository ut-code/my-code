### 文字列（str）

文字列はシングルクォート (`'`) またはダブルクォート (`"`) で囲んで作成します。

文字列の連結は `+` 演算子、繰り返しは `*` 演算子を使います。

```python-repl
>>> name = "Guido"
>>> greeting = 'Hello'
>>> full_greeting = greeting + ", " + name + "!"
>>> print(full_greeting)
Hello, Guido!
>>> print("-" * 10)
----------
```
