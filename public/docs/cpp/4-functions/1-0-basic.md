## 関数の宣言と定義

PythonやJavaScriptのような言語では、関数をどこに書いても（あるいは実行時に解決されて）呼び出せることが多いですが、C++のコンパイラはコードを上から下へと一直線に読みます。そのため、**「使用する前に、その関数が存在すること」**をコンパイラに知らせる必要があります。
