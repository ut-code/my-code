### `if`文

`if (条件式)` の条件式は `bool` に変換可能なものである必要があります。C++では `0` は `false`、それ以外は `true` とみなされます。

```cpp:control-if.cpp
#include <iostream>

int main() {
    // --- if文 ---
    const int score = 85;
    if (score >= 90) {
        std::cout << "Grade: A" << std::endl;
    } else if (score >= 80) {
        std::cout << "Grade: B" << std::endl;
    } else {
        std::cout << "Grade: C or below" << std::endl;
    }

    return 0;
}
```

```cpp-exec:control-if.cpp
Grade: B
```
